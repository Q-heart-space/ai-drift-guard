# Contributing / 参与指引

Thanks for your interest in AI-Drift-Guard.

## How to give feedback

- Open a [GitHub Issue](https://github.com/Q-heart-space/ai-drift-guard/issues)
- Pick the appropriate template: **Bug report** or **Feature request**
- Include a concrete example: what you said, what the agent did, and what you expected.

## Scope

This repository contains only the portable version of the protocol.  
Internal Q博士-specific governance tooling is not published here.

## Code of conduct

Be constructive, be specific, and remember the protocol itself is a self-check: if you spot a rule the maintainer is violating, file it as a bug.
